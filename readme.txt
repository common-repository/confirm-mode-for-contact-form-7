This is an add-on to introduce confirmation mode into Contact Form 7.


== Description ==

This plugin is an add-on to the Contact Form 7 plugin.

It changes the form submission procedure to the following three steps.
  1) input
  2) confirmation
  3) send

Contact Form 7 defaults to the following two steps.
  1) input
  2) send


== Frequently Asked Questions ==

= Which Contact Form 7 versions are supported? =

Confirmed with Contact Form 7 version 5.9.8.

= What browsers are supported? =

Confirmed on Google Chrome 128 on Windows 11.
